# Created By Me EVM Api

## Description
This application serves as an example API for the Ethereum Virtual Machine (EVM) contract set for the Created By Me application.

## Setup

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
